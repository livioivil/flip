Welcome to the dev-version of the library flip on CRAN

To install this github version type (in R):

install.packages("devtools") if you dont already have it.

library(devtools)

install_github("flip", "livioivil")

Some examples

library(flip)
set.seed(1)
y=rnorm(10)+.5
res=flip(y)
summary(res)
##  Call:
##  flip(Y = y) 
## 1023 permutations.
##   Test  Stat tail p-value sig.
## Y    t 2.561   ><  0.0293    *
plot(res)
